# tashi-frontend

### Live website : https://tashi.netlify.com/ 

## GETTING STARTED

- Fork or clone repository 
- Use `yarn create react-app <name>` to create new application
- Install these dependencies using `yarn add` or `npm install`:
  - `react-redux`
  - `redux`
  - `axios`
  - `redux-logger`
  - `redux-thunk`
  - `react-router`
  - `react-router-dom`
